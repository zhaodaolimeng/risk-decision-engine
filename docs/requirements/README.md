## MVP 核心功能模块

```
风险决策引擎 (MVP)
├── 规则引擎模块
│   ├── 规则定义（JSON配置）
│   ├── 规则管理（CRUD）
│   └── 规则执行
├── 决策流模块
│   ├── 流程定义（JSON配置）
│   ├── 流程编排
│   └── 流程执行
├── 数据管理模块
│   ├── 数据源接入（HTTP API）
│   └── 数据校验
├── 发布管理模块
│   └── 策略发布（全量发布）
├── 审计日志模块
│   ├── 操作日志
│   └── 决策日志
└── 权限管理模块（简化版）
    └── 简单权限控制
```

## MVP 业务流程简述

1. **贷款申请** → 用户提交贷款申请
2. **数据采集** → HTTP API 拉取数据
3. **数据校验** → 基础数据校验
4. **规则引擎** → 硬规则过滤
5. **决策输出** → 通过/拒绝/人工复核
6. **日志记录** → 决策日志保存
