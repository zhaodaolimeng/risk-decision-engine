# 风险决策引擎 - 接口需求

## 1. 接口概述

### 1.1 通信协议

- 协议：HTTPS
- 数据格式：JSON
- 字符编码：UTF-8
- 签名算法：HMAC-SHA256

### 1.2 通用响应格式

```json
{
  "code": "0000",
  "message": "成功",
  "requestId": "req_20240101120000_abc123",
  "timestamp": 1704067200000,
  "data": {}
}
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| code | STRING | 是 | 响应码，"0000"表示成功 |
| message | STRING | 是 | 响应消息 |
| requestId | STRING | 是 | 请求唯一标识 |
| timestamp | LONG | 是 | 响应时间戳（毫秒） |
| data | OBJECT | 否 | 响应数据 |

### 1.3 错误码

| 错误码 | 说明 |
|--------|------|
| 0000 | 成功 |
| 1001 | 参数错误 |
| 1002 | 签名验证失败 |
| 2001 | 策略不存在 |
| 2002 | 策略未启用 |
| 3001 | 数据源调用失败 |
| 3002 | 数据缺失 |
| 4001 | 系统内部错误 |
| 4002 | 系统繁忙 |
| 4003 | 超时 |

## 2. 决策接口

### 2.1 同步决策接口

#### 接口描述

执行风险决策，同步返回决策结果。

#### 基本信息

| 项目 | 内容 |
|------|------|
| URL | /api/v1/decision/execute |
| Method | POST |
| Content-Type | application/json |

#### 请求参数

```json
{
  "requestId": "req_20240101120000_abc123",
  "productId": "P001",
  "strategyId": "S001",
  "businessId": "biz_1234567890",
  "callbackUrl": "https://example.com/callback",
  "data": {
    "applicant": {
      "name": "张三",
      "idCard": "110101199001011234",
      "phone": "13800138000",
      "age": 34,
      "education": "本科",
      "monthlyIncome": 15000
    },
    "application": {
      "applyAmount": 100000,
      "applyTerm": 12,
      "loanPurpose": "消费",
      "repaymentType": "等额本息"
    },
    "device": {
      "deviceFingerprint": "dev_abc123xyz",
      "deviceType": "iOS",
      "ipAddress": "192.168.1.1"
    }
  },
  "extensions": {}
}
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| requestId | STRING | 是 | 请求唯一标识 |
| productId | STRING | 是 | 产品ID |
| strategyId | STRING | 否 | 策略ID，不传使用产品默认策略 |
| businessId | STRING | 是 | 业务流水号 |
| callbackUrl | STRING | 否 | 回调地址，异步决策时使用 |
| data | OBJECT | 是 | 业务数据 |
| extensions | OBJECT | 否 | 扩展字段 |

#### 响应参数

```json
{
  "code": "0000",
  "message": "成功",
  "requestId": "req_20240101120000_abc123",
  "timestamp": 1704067200000,
  "data": {
    "decisionId": "dec_20240101120000_xyz789",
    "businessId": "biz_1234567890",

          "result": "PASS",
          "executeTime": 5
        }
      ],
      "scorecardResults": [
        {
          "scorecardId": "SC001",
          "scorecardName": "信用评分卡",
          "score": 720,
          "dimensionScores": [
            {
              "dimension": "基本信息",
              "score": 150,
              "weight": 0.2
            }
          ]
        }
      ],

        {
          "modelId": "M001",

```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| decisionId | STRING | 是 | 决策ID |
| businessId | STRING | 是 | 业务流水号 |
| decision | STRING | 是 | 决策结果：APPROVE/REJECT/MANUAL |
| decisionCode | STRING | 是 | 决策码 |
| decisionReason | STRING | 是 | 决策原因 |
| score | INTEGER | 否 | 信用评分 |
| riskLevel | STRING | 否 | 风险等级：LOW/MEDIUM/HIGH |
| approvalAmount | DECIMAL | 否 | 审批额度 |
| approvalTerm | INTEGER | 否 | 审批期限 |
| interestRate | DECIMAL | 否 | 利率 |
| executeDetail | OBJECT | 否 | 执行详情（可配置是否返回） |
| suggestions | ARRAY | 否 | 建议 |
| tags | ARRAY | 否 | 标签 |
| executeTime | INTEGER | 是 | 执行耗时（毫秒） |

### 2.2 异步决策接口

#### 接口描述

提交异步决策请求，结果通过回调返回。

#### 基本信息

| 项目 | 内容 |
|------|------|
| URL | /api/v1/decision/executeAsync |
| Method | POST |
| Content-Type | application/json |

#### 请求参数

同同步决策接口，`callbackUrl` 必填。

#### 响应参数

```json
{
  "code": "0000",
  "message": "成功",
  "requestId": "req_20240101120000_abc123",
  "timestamp": 1704067200000,
  "data": {
    "decisionId": "dec_20240101120000_xyz789",
    "businessId": "biz_1234567890",
    "status": "PROCESSING"
  }
}
```

#### 回调接口

业务方需实现的回调接口：

```json
POST {callbackUrl}
{
  "decisionId": "dec_20240101120000_xyz789",
  "businessId": "biz_1234567890",
  "status": "COMPLETED",
  "result": {
    // 同同步决策接口的 data 部分
  },
  "timestamp": 1704067200000
}
```

### 2.3 决策结果查询接口

#### 接口描述

根据决策ID查询决策结果。

#### 基本信息

| 项目 | 内容 |
|------|------|
| URL | /api/v1/decision/query |
| Method | GET |

#### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| decisionId | STRING | 二选一 | 决策ID |
| businessId | STRING | 二选一 | 业务流水号 |

#### 响应参数

同同步决策接口响应。
