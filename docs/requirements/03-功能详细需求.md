# 风险决策引擎 - 功能详细需求

## 1. 规则引擎模块

### 1.1 规则定义

#### 1.1.1 规则类型

| 类型 | 描述 | 示例 |
|------|------|------|
| 布尔规则 | 返回 true/false | 年龄 < 18 岁 拒绝 |
| 数值规则 | 返回数值结果 | 计算收入负债比 |
| 评分规则 | 返回评分 | 收入分段评分 |
| 枚举规则 | 返回枚举值 | 风险等级：高/中/低 |

#### 1.1.2 规则结构

```yaml
rule:
  id: "R001"
  name: "年龄准入规则"
  description: "申请人年龄必须在18-60岁之间"
  type: "BOOLEAN"
  condition:
    operator: "AND"
    expressions:
      - field: "age"
        operator: ">="
        value: 18
      - field: "age"
        operator: "<="
        value: 60
  actions:
    true: "PASS"
    false: "REJECT"
    falseReason: "年龄不符合要求"
  priority: 100
  version: "1.0"
  status: "ACTIVE"
```

### 1.2 规则操作符

#### 1.2.1 比较操作符

| 操作符 | 描述 | 支持类型 |
|--------|------|----------|
| == | 等于 | 所有类型 |
| != | 不等于 | 所有类型 |
| > | 大于 | 数值、日期 |
| >= | 大于等于 | 数值、日期 |
| < | 小于 | 数值、日期 |
| <= | 小于等于 | 数值、日期 |
| IN | 在集合中 | 所有类型 |
| NOT_IN | 不在集合中 | 所有类型 |
| CONTAINS | 包含 | 字符串 |
| STARTS_WITH | 开头 | 字符串 |
| ENDS_WITH | 结尾 | 字符串 |
| MATCHES | 正则匹配 | 字符串 |
| IS_NULL | 为空 | 所有类型 |
| IS_NOT_NULL | 不为空 | 所有类型 |

#### 1.2.2 逻辑操作符

| 操作符 | 描述 |
|--------|------|
| AND | 与 |
| OR | 或 |
| NOT | 非 |

### 1.3 规则管理功能

| 功能 | 描述 |
|------|------|
| 规则创建 | 可视化规则编辑器 |
| 规则编辑 | 修改规则内容，版本管理 |
| 规则复制 | 基于已有规则创建新规则 |
| 规则删除 | 逻辑删除，保留历史 |
| 规则搜索 | 按ID、名称、标签等搜索 |
| 规则导入/导出 | JSON/YAML格式 |
| 规则测试 | 单规则测试 |

### 3.2 决策流示例

```
START
  │
  ▼
┌─────────────┐
│ 规则集: 准入 │
│ (黑名单/    │
│  年龄/身份) │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ 评分卡: 信用 │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ 模型: 违约预测│
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ 决策矩阵    │
└──────┬──────┘
       │
   ┌───┴───┐
   │       │
   ▼       ▼
  PASS   REJECT
   │       │
   └───┬───┘
       │
       ▼
      END
```

### 4.2 特征管理

#### 4.2.1 特征类型

| 类型 | 描述 | 示例 |
|------|------|------|
| 原始特征 | 直接从数据源获取 | age、income |
| 衍生特征 | 基于原始特征计算 | incomeDebtRatio = income / debt |
| 聚合特征 | 时间窗口聚合 | recent30DaysQueryCount |

#### 4.2.2 特征定义

```yaml
feature:
  name: "incomeDebtRatio"
  displayName: "收入负债比"
  description: "月收入与月负债的比率"
  type: "DOUBLE"
  dataSource: "derived"
  expression: "income / monthlyDebt"
  dependencies:
    - "income"
    - "monthlyDebt"
  validations:
    - min: 0
    - max: 100
  tags:
    - "还款能力"
    - "衍生特征"
```

## 6. 发布管理模块

### 6.1 发布策略

| 策略 | 描述 | 适用场景 |
|------|------|----------|
| 全量发布 | 一次性切换所有流量 | 紧急修复、小改动 |
| 灰度发布 | 逐步放量 | 新策略上线 |
| 蓝绿发布 | 两套环境切换 | 重大变更 |
| 回滚 | 快速回退到上一版本 | 发布出现问题 |

### 6.2 发布流程

1. **创建发布单** - 选择要发布的策略版本
2. **变更检查** - 自动检查变更影响
3. **审批** - 负责人审批
4. **执行发布** - 按策略执行发布
5. **监控观察** - 实时监控指标
6. **完成/回滚** - 根据监控结果决定
