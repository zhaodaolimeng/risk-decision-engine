# 风险决策引擎 - 数据需求

## 1. 数据分类

### 1.1 按数据来源分类

| 分类 | 描述 | 示例 |
|------|------|------|
| 申请数据 | 用户填写的申请信息 | 姓名、身份证、手机号、收入、学历等 |
| 征信数据 | 央行及第三方征信数据 | 征信报告、逾期记录、查询记录等 |
| 行为数据 | 用户历史行为数据 | 消费记录、还款记录、登录行为等 |
| 外部数据 | 第三方数据源 | 运营商数据、社保公积金、司法信息等 |
| 设备数据 | 设备相关信息 | 设备指纹、IP地址、地理位置等 |

## 3. 衍生特征

### 3.1 负债相关

| 特征名 | 特征标识 | 计算逻辑 |
|--------|----------|----------|
| 月负债合计 | monthlyDebtTotal | Σ各贷款月还款额 + Σ信用卡最低还款额 |
| 收入负债比 | incomeDebtRatio | 月负债合计 / 月收入 |
| 债务收入比 | debtIncomeRatio | 总负债余额 / (月收入 * 12) |
| 信用卡使用率 | creditCardUtilization | 信用卡已用额度 / 信用卡授信总额 |
| 授信倍数 | creditMultiplier | 总授信额度 / 月收入 |

### 3.2 征信查询相关

| 特征名 | 特征标识 | 计算逻辑 |
|--------|----------|----------|
| 查询频率 | queryFrequency | 近N个月查询次数 / N |
| 贷款审批查询占比 | loanQueryRatio | 贷款审批查询次数 / 总查询次数 |
| 信用卡审批查询占比 | cardQueryRatio | 信用卡审批查询次数 / 总查询次数 |

### 3.3 历史行为相关

| 特征名 | 特征标识 | 计算逻辑 |
|--------|----------|----------|
| 通过率 | approvalRate | 历史通过次数 / 历史申请次数 |
| 逾期率 | overdueRate | 历史逾期笔数 / 历史借款笔数 |
| 账户龄 | accountAge | 当前日期 - 注册日期 |
| 最近借款距今天数 | daysSinceLastLoan | 当前日期 - 最近借款日期 |

## 4. 数据安全要求

### 4.1 数据加密

- **敏感字段加密存储**：身份证号、手机号、姓名等
- **传输加密**：HTTPS/TLS
- **密钥管理**：定期轮换，安全存储

### 4.2 数据脱敏

| 字段 | 脱敏方式 | 示例 |
|------|----------|------|
| 姓名 | 中间字符替换 | 张* |
| 手机号 | 中间4位替换 | 138****1234 |
| 身份证号 | 中间8位替换 | 110***********1234 |
| 地址 | 详细地址隐藏 | 北京市朝阳区**** |
| 银行卡号 | 中间12位替换 | 6222 **** **** 1234 |

### 4.3 数据留存

- **决策日志**：至少留存5年
- **个人信息**：用户注销后按法规要求删除
- **审计日志**：至少留存3年
